# 🎯 MULTI-TIMEFRAME TREND EXAMPLES

## 📊 HOW IT WORKS

```
Divergence Timeframe → Check Trend On
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1M (Monthly)         → 1d (Daily)
1w (Weekly)          → 4h (4-hour)
1d (Daily)           → 1h (1-hour)
4h (4-hour)          → 15m (15-minute)
1h (1-hour)          → 5m (5-minute)
```

**Why this is powerful:**
- Higher TF = Big picture (divergence structure)
- Lower TF = Current momentum (is it starting NOW?)
- Both must align = High probability trade
